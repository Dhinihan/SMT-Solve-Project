Find if the inequation is satisfiable and give one answer if possible.

The inequation is in the form of A0 + A[i,j]*X[i,j] >= 0. where i goes from 1 to w and j goes from 1 to n. The domain of the x in the output is (X[i,j] = 0 or X[i,j] = 1). For each i exists only one j that X[i,j] = 1. the response R[0] is equal to 1 if it is satisfiable and the values for each X[i,j] is in the form X[i,j] = R[(i-1)*n + j]. If not satisfiable, R[0] = 0.
